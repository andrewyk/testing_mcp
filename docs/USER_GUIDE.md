# TodoApp User Guide

Welcome to TodoApp! This guide will help you get started with managing your tasks efficiently.

## Getting Started

### Creating an Account

1. Navigate to the TodoApp homepage
2. Click on "Register" 
3. Fill in your information:
   - Name
   - Email address
   - Password (minimum 6 characters)
4. Click "Register" to create your account
5. You'll be automatically logged in

### Logging In

1. Go to the login page
2. Enter your email and password
3. Click "Login"
4. You'll be redirected to your dashboard

## Managing Tasks

### Creating a New Task

1. Click the **"Add Task"** button (or press `Ctrl+K`)
2. Fill in the task details:
   - **Title** (required): Brief description of your task
   - **Description** (optional): Add more details about the task
   - **Priority**: Choose Low, Medium, or High
   - **Due Date**: Set a deadline for your task
   - **Project**: Assign the task to a project
   - **Tags**: Add relevant tags for categorization
3. Click **"Create"** to save your task

### Editing a Task

1. Click the **edit icon** (pencil) on any task
2. Modify the task details
3. Click **"Update"** to save changes

### Completing a Task

- Click the **checkbox** next to a task to mark it as complete
- Click again to mark it as incomplete
- Completed tasks are shown with a strikethrough

### Deleting a Task

1. Click the **trash icon** on a task
2. Confirm the deletion
3. The task will be moved to trash (soft delete)
4. To permanently delete, add the feature in settings

## Organizing Tasks

### Using Projects

Projects help you group related tasks together.

**Creating a Project:**
1. Look for the Projects section in the sidebar
2. Click the "+" button
3. Enter project name, description, and choose a color
4. Click "Create"

**Filtering by Project:**
- Click on a project name in the sidebar
- Only tasks in that project will be displayed

### Using Tags

Tags provide flexible categorization for your tasks.

**Creating a Tag:**
1. Go to the Tags section
2. Create a new tag with a name and color
3. Apply tags when creating or editing tasks

**Filtering by Tags:**
- Click on a tag to see all tasks with that tag
- You can apply multiple tags to a single task

## Filtering and Sorting

### Quick Filters

Use the sidebar to quickly filter tasks:
- **All Tasks**: View all your tasks
- **Active**: Show only incomplete tasks
- **Completed**: Show only completed tasks
- **High Priority**: Show only high-priority tasks

### Advanced Filtering

Combine filters for more specific views:
- Filter by status (active/completed)
- Filter by priority level
- Filter by project
- Filter by due date (today, this week, overdue)

### Sorting Tasks

Sort your tasks by:
- **Created Date**: When the task was created
- **Due Date**: Upcoming deadlines first
- **Priority**: High priority tasks first
- **Title**: Alphabetical order

Change sort direction from ascending to descending.

### Search

1. Use the search box to find specific tasks
2. Searches through task titles and descriptions
3. Results update as you type

## Dashboard and Statistics

Your dashboard shows important metrics:

- **Total Tasks**: All your tasks
- **Active**: Tasks not yet completed
- **Completed**: Finished tasks
- **Today**: Tasks due today
- **This Week**: Tasks due in the next 7 days
- **Overdue**: Tasks past their due date

These statistics help you track your productivity and identify tasks that need attention.

## Keyboard Shortcuts

Use these shortcuts for faster navigation:

- `Ctrl/Cmd + K`: Quick add new task
- `Escape`: Close modals
- Click checkbox: Toggle task completion
- Arrow keys: Navigate between tasks (when focused)

## Dark Mode

Toggle between light and dark themes:

1. Click the **moon/sun icon** in the header
2. Your preference is saved automatically
3. The app will remember your choice next time you visit

## Tips for Productivity

### Best Practices

1. **Use the Inbox**: Create tasks quickly and organize them later
2. **Set Priorities**: Mark important tasks with high priority
3. **Use Due Dates**: Set realistic deadlines to stay on track
4. **Break Down Large Tasks**: Use subtasks for complex projects
5. **Review Regularly**: Check your dashboard daily
6. **Archive Completed Tasks**: Keep your active list clean

### Project Organization

- Create projects for different areas of your life (Work, Personal, Learning)
- Use colors to visually distinguish projects
- Keep project names short and descriptive

### Tag Strategy

- Create tags for contexts (e.g., @home, @office, @phone)
- Use tags for energy levels (e.g., quick-win, deep-work)
- Tag recurring themes (e.g., meeting, research, writing)

### Priority Management

**High Priority:**
- Urgent and important tasks
- Tasks with imminent deadlines
- Critical work items

**Medium Priority:**
- Important but not urgent
- Regular work tasks
- Tasks with flexible deadlines

**Low Priority:**
- Nice-to-have tasks
- Long-term goals
- Ideas for future consideration

## Managing Multiple Projects

### Project-Based Workflow

1. Create a project for each major area of focus
2. Break down projects into actionable tasks
3. Use tags within projects for additional organization
4. Set project-level goals and track progress

### Cross-Project Tasks

Some tasks may relate to multiple projects:
- Assign the task to the primary project
- Use tags to reference other related projects
- Add notes in the description to link projects

## Troubleshooting

### Task Not Saving

- Check your internet connection
- Ensure all required fields are filled
- Try refreshing the page and logging in again

### Cannot See Tasks

- Check your current filter settings
- Make sure you're not filtering by a specific project or status
- Use "All Tasks" filter to see everything

### Login Issues

- Verify your email and password
- Check caps lock is off
- Use password reset if needed (feature to be added)

### Performance Issues

- Clear your browser cache
- Update to the latest browser version
- Check if you have too many browser extensions

## Privacy and Security

### Data Protection

- All passwords are securely hashed
- Communication is encrypted (when using HTTPS)
- Your data is private and not shared with third parties

### Account Security

- Use a strong, unique password
- Don't share your login credentials
- Log out when using shared devices
